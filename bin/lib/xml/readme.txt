This folder should be used for project xml files. The folder MUST contain the project
config.xml file as LaunchPad uses this to configure the framework and import your assets.

You can add other xml files in to this folder, and declare them as assets in the config.xml
file, making them available at any point within your project.

You can also utilise the com.thirdsense.utils.XMLLoader class if you wish, or use your own
method.
