This folder should be used for assets with the 3rd extension. Typically, a 3rd file is
essentially a .zip archive containing asset files such as mp3, jpg, gif, png, and swf.

The files located here should be referenced in the ../xml/config.xml file for use with the
LaunchPad framework.
