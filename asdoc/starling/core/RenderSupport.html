Packagestarling.core
Classpublic class RenderSupport
InheritanceRenderSupport Object

A class that contains helper methods simplifying Stage3D rendering. A RenderSupport instance is passed to any "render" method of display objects. It allows manipulation of the current transformation matrix (similar to the matrix manipulation methods of OpenGL 1.x) and other helper methods.

public function batchQuadBatch(quadBatch:QuadBatch, parentAlpha:Number):void

Adds a batch of quads to the current batch of unrendered quads. If there is a state change, all previous quads are rendered at once.

Note that you should call this method only for objects with a small number of quads (we recommend no more than 16). Otherwise, the additional CPU effort will be more expensive than what you save by avoiding the draw call.

pushClipRect()method 
public function pushClipRect(rectangle:Rectangle):Rectangle

The clipping rectangle can be used to limit rendering in the current render target to a certain area. This method expects the rectangle in stage coordinates. Internally, it uses the 'scissorRectangle' of stage3D, which works with pixel coordinates. Any pushed rectangle is intersected with the previous rectangle; the method returns that intersection.

Parameters

rectangle:Rectangle
