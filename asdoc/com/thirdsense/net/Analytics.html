trackEvent()method 
public static function trackEvent(category:String, action:String, label:String, value:Number = 0):Boolean

Reports to Google Analytics that an event has occured. Handy for tracking in-app stats.

Parameters

category:String — The category of the event. Best used to pass through the platform (eg. "AppIOS", "AppAndroid", "AppWeb" etc.)
 
action:String — The name of the event that has occured (eg. "GameLoaded", "ScoreSubmitted" etc.)
 
label:String — If a value is to be associated, this is the label to use for it (eg. "QuizQuestionAnsweredCorrect" )
 
value:Number (default = 0) — The value of the label. (eg 1). GA increments source values by this value amount

Returns
Boolean — A boolean value that indicates if the call was made successfully
