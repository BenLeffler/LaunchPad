init()method 
public static function init(onComplete:Function = null, force_new_login:Boolean = false, auto_login_only:Boolean = false, browserInvoke:Function = null):void

Initializes a connection to the Facebook API and begins the login process. Facebook permissions are set up in the LaunchPad config.xml file.

Parameters

onComplete:Function (default = null) — Function that gets called upon a result of this process
 
force_new_login:Boolean (default = false) — Pass as true if you wish to force a new Facebook login (instead of using an existing session)
 
auto_login_only:Boolean (default = false) — Pass as true if you only wish to use an existing saved session
 
browserInvoke:Function (default = null) — If you wish to use a custom browser window, pass the browser invoke function. This function MUST return a StageWebView object.
