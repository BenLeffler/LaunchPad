Loads an image from a specified url

Parameters

myURL:String — The url to load the image from
 
target:DisplayObjectContainer — The target container to place a bitmap copy of the loaded image
 
maxwidth:Number (default = 0) — The maximum width of the image. Leave as 0 for the native size to be used
 
maxheight:Number (default = 0) — The maximum height of the image. Leave as 0 for the native size to be used
 
scale_type:String (default = constrain) — Determines the scale type to be used. Defaults to SmoothImageLoad.CONSTRAIN.
 
requiresPolicyFile:Boolean (default = false) — If a crossdomain load is required, pass this as true and ensure you have loaded the appropriate context security policy first.
 
cacheImage:Boolean (default = false) — If the image should be cached for later use in the session, pass this as true. Defaults to false.
