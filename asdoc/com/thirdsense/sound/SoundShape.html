apply()method
public static function apply(channel:SoundChannel, shape:String, frames:int = 60, onShapeComplete:Function = null):void

Applies a tween based sound effect to a SoundChannel object

Parameters

channel:SoundChannel — The SoundChannel object to apply the transformation to
 
shape:String — The type of effect to apply. (eg. SoundShape.FADE_IN, SoundShape.PAN_RIGHT)
 
frames:int (default = 60) — The number of frames to apply this transformation over
 
onShapeComplete:Function (default = null) — The function that is called upon the transformation having been completed
