init()method 
public function init(target:DisplayObject, type:String, viewPort:Rectangle = null):void

Initializes a scroll control instance against a desired Starling display object. If the target is removed from the stage at any point, the scroll control instance self-terminates.

Parameters

target:DisplayObject — The Starling display object to register this controller against
 
type:String — The type of control to use
 
viewPort:Rectangle (default = null) — The viewport of the scroll area. If passed as null, the entire stage is used as the viewport
