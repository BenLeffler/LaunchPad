public static function addAlert(heading:String, copy:String, element:MovieClip = null, onComplete:Function = null):Boolean

Adds an Alert UI element to the stage. Only a single instance of any UI element may exist at any point

Parameters

heading:String — Text that is displayed in the heading field
 
copy:String — Text that is displayed in the copy field
 
element:MovieClip (default = null) — The MovieClip to use as the prompt scaffold. If left as null, a GenericAlert instance is created and used
 
onComplete:Function (default = null) — The function to call when the user dismisses the alert

Returns
Boolean — A Boolean value that indicates if the UI element was added to the stage
