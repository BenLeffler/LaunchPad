public function init(target:DisplayObject, speed:Number = 5, onTouch:Function = null):void

Initialises the drag control object to start working with a desired Starling DisplayObject

Parameters

target:DisplayObject — The target DisplayObject to attach the DragControl object functionality to
 
speed:Number (default = 5) — The speed of the drag. The higher this number, the slower the speed. For 1:1 dragging, pass this as 1.
 
onTouch:Function (default = null) — The function that is called whenever a TouchPhase.BEGAN, TouchPhase.MOVED, or TouchPhase.ENDED phase is detected on the object. The function must accept a Touch object as a param.
