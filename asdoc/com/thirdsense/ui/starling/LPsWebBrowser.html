Method Detail
create()method
public static function create(url:String, redirect_url:String):*

Creates a new instance of the LPsWebBrowser. If a url is passed through, a browser session is automatically added to the root Starling container. If url is left as a blank string, the static singleton is only created and allows you to customize callback functions prior to invoking the browser.

Parameters

url:String — The url to immediately load with a new browser session
 
redirect_url:String — If the browser session detects a location other than the one passed in the url parameter, the browser will redirect to this location

Returns
* — The LPsWebBrowser singleton instance

invoke()method 
public static function invoke():*

Invokes an LPsWebBrowser session. If a singleton doesn't exist prior to calling invoke a new instance is created. This method is mainly used for classes such as FacebookInterface which can invoke a browser session to prompt the user for an oAuth login.

Returns
* — The StageWebView instance which is summarily created as part of the browser session invoke.
