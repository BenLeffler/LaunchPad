LPsCheckList()Constructor
public function LPsCheckList(texture_packs:Array, defaults:Array = null, multichoice:Boolean = false, onToggle:Function = null)

Creates a new CheckList element

Parameters
texture_packs:Array — An array of texture_packs to use with this checklist. Each element is assigned an integer id (starting with 0)
 
defaults:Array (default = null) — An array of integer id's that are to start as marked as 'selected on'. If the multichoice param is 'false', the first id in this array (or id:0 if this is passed as null) will be selected
 
multichoice:Boolean (default = false) — Set as true if the user is allowed to have multiple (or none) selections within this element
 
onToggle:Function (default = null) — A function that can be called when a user changes a selection
