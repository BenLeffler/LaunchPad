create()method
public static function create(clip:MovieClip, timeline:MovieClip = null, start_frame:int = 0, end_frame:int = 0, crop:Rectangle = null, filter_pad:int = 5, onNewFrame:Function = null, onNewFrameArgs:Array = null):SpriteSequence

Creates a sprite sequence object from a MovieClip by analysing and taking a bitmapdata snapshot of each frame, placing each in to a vector for use in blitting and texture packing in Starling

Parameters

clip:MovieClip — The movieclip to capture frame-by-frame
 
timeline:MovieClip (default = null) — If a sub-clip needs it's timeline to advance with the clip timeline, pass it through in this parameter
 
start_frame:int (default = 0) — The number of the first frame
 
end_frame:int (default = 0) — The number of the final frame. Leaving this as 0 will capture the every frame in the clip
 
crop:Rectangle (default = null) — A rectangle object that will act as a clipping box for the resulting texture atlas
 
filter_pad:int (default = 5) — Allows a 'bleed' area around the movieclip bounds to account for any filters that may exceed the bounds area (like a glow filter or drop shadow for example)
 
onNewFrame:Function (default = null) — A function to call as the playhead advances through the clip timeline. If you have clip children that require calculations to be made on each frame, you can use this method.
 
onNewFrameArgs:Array (default = null) — An array of arguments to pass through to onNewFrame function.

Returns
SpriteSequence — The resulting spritesequence object
