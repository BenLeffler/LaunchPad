loadLibrary()method 
public static function loadLibrary(id_array:Array, onComplete:Function = null, preloader:MovieClip = null):void

Starts a post load procedure of LaunchPad library assets

Parameters

id_array:Array — An array of library id's to load one-by-one
 
onComplete:Function (default = null) — The callback function once the procedure has reached it's end
 
preloader:MovieClip (default = null) — The preloader ui element to use. If left as null, the LaunchPad generic loader element is used. For no loader bar, pass this as an empty movieclip.
